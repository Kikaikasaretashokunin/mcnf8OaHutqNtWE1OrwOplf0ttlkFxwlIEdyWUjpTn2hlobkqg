2.6 Resumen.

Las sentencias de control de un programa son instrucciones para distintos propósitos, como:

  • Programación de variables (o parámetros).
  • Control de la ejecución de bloques
  • Creación de subrutinas.
  • Personalización de programas.

Todas estas sentencias pertenecen a lo que se llama programación de alto nivel, su programación, se realiza escribiendo estas sentencias entre paréntesis. La programación de alto nivel rebasa los objetivos del presente curso, por ello solo se incluyen aquellas sentencias más útiles para el nivel de conocimientos de este curso.

Las sentencias de control de programa que se estudian son:

  • Sentencias de salto en un programa. (GOTO N[Expresión])
  • Sentencias de repetición. (RPT N[expresión],N[expresión]) N[expresión]
  • Sentencias de creación de una subrutina. (SUB N) - (RET)
  • Sentencias de llamada a una subrutina. (CALL N)
